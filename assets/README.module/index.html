    name¶

    name content type for Tide distribution for Drupal 8

    Tide is a Drupal 8 distribution focused on delivering an API first, headless Drupal content administration site.

    What is in this package¶

    • content type
    • fields
    • blocks
    • views
    • JSONAPI module integration

    Installation¶

    To install this package, add this custom repository to repositories section of your composer.json:

    {
  "repositories": {        
      "dpc-sdp/tide_page": {
          "type": "vcs",
          "no-api": true,
          "url": "https://github.com/dpc-sdp/_machine_name_.git"
      }
  }
}

    Require this package as any other Composer package: 

    composer require dpc/_machine_name_
